**그래프 최단거리(BFS)**

다음 그래프에서 1번 정점에서 각 정점으로 가는 최소 이동 간선수를 출력하세요.

![img.png](img.png)

**입력**

첫째 줄에는 정점의 수 N(1<=N<=20)와 간선의 수 M가 주어진다. 그 다음부터 M줄에 걸쳐 연결정보가 주어진다.

**출력**

1번 정점에서 각 정점으로 가는 최소 간선수를 2번 정점부터 차례대로 출력하세요.

**Cases**

6 9<br>
1 3<br>
1 4<br>
2 1<br>
2 5<br>
3 4<br>
4 5<br>
4 6<br>
6 2<br>
6 5<br>
output:<br>
2 : 3<br>
3 : 1<br>
4 : 1<br>
5 : 2<br>
6 : 2
